Retrieve Table With Bulk Requests
---------------------------------

See:

* :py:func:`puresnmp.bulktable`

Python Code
~~~~~~~~~~~

.. code-block:: python

    from pprint import pprint
    from puresnmp import bulktable

    IP = '127.0.0.1'
    COMMUNITY = 'private'
    OID = '1.3.6.1.2.1.2.2'

    result = bulktable(IP, COMMUNITY, OID)
    pprint(result)
